# Changelog

## 2.1.0+4 (2026-04-28)

### Added
- 新增运单同一运单内「同产品 + 同批号」重复添加拦截与提醒。
- 订单列表状态新增「全部」并默认选中，增加「完成/未完成」数量统计。
- 出库日历支持点击运单后按运单过滤当日出库明细。
- 二维码预览页新增二维码大小滑条，并持久化保存大小设置。

### Changed
- 新增运单产品下拉改为按库存数量降序展示。
- 新增运单 TS 展示改为基于基础资料 TS 勾选结果显示标签。
- 出库日历订单摘要增强为「订单 X单 · Y箱」，并展示运单号、商家、出库箱数。
- 出库明细增强展示运单号与商家信息，便于追溯当日出库来源。
- 出库变化文案调整为「库存变化 -X箱」，且仅在变化大于 0 时显示。
- 日历快捷筛选与「自定义范围」按钮固定单行（窄屏横向滚动，不再自动换行）。

### Fixed
- 修复新增运单产品信息 TS 识别与展示不一致的问题。
- 修复出库日历窄屏下筛选控件换行导致的布局问题。
- 修复二维码预览在较小可视区域的内容溢出问题。
