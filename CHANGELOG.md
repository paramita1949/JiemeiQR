# Changelog

## 3.1.5 (2026-04-29)

### Changed
- 首页库存区恢复并优化“库存变化”展示：不再单独占一整行，改为集成在「在途货物」卡片内作为次级信息（如 `库存变化 -6000箱`），减少空间占用并保持信息层级清晰。

## 3.1.4 (2026-04-29)

### Changed
- 运单详情「编辑产品明细」弹窗交互调整：`箱数`输入与`当前批号可用 XXX 箱`提示上移到批号列表上方，减少来回滚动和视线跳转。
- 编辑批号选择列表接入同规则高亮：同产品下若存在“同日期、多批号”，批号中差异字母会高亮显示，便于快速识别差异批号。

## 3.1.3 (2026-04-29)

### Changed
- 出库日历「出库明细」数量展示改为仅显示箱数，不再显示“箱数·板数”折算信息。

## 3.1.2 (2026-04-29)

### Changed
- 批号差异字母高亮规则全局统一：仅当“同产品 + 同日期”存在多个不同批号时触发高亮。
- 运单详情、库存明细、订单信息备货汇总、新增/编辑运单（批号下拉与已添加明细）、出库日历全部接入同一高亮逻辑。
- 出库日历明细行展示升级为“产品编码 · 批号 · 日期”，并修复窄屏下可能的行内溢出。

### Added
- 新增出库日历批号高亮测试，覆盖同产品同日期多批号差异字母高亮场景。

## 3.1.1 (2026-04-29)

### Changed
- 运单详情产品行标题样式调整：产品编号恢复常规颜色，仅日期保持红色。
- 同日期多批号继续使用“批号字母差异高亮”规则，便于快速识别差异位。
- 产品行标题改为单行显示并启用超长省略，避免换行导致信息错位。

### Added
- 新增库存同步审计测试：覆盖“删除单条明细后预占即时更新”“删除已完成订单后库存与出库流水同步恢复”。

## 3.0.15 (2026-04-29)

### Fixed
- 修复运单详情中“删除产品明细确认后无反馈”的问题：删除成功会即时刷新并提示，已完成订单会明确提示不允许删单条明细。

### Added
- 运单详情每条产品明细新增编辑按钮，支持直接修改批号与箱数并保存。

## 3.0.14 (2026-04-29)

### Added
- 运单详情每条产品明细新增编辑入口，可修改批号与箱数并即时保存。

### Changed
- 新增运单「产品明细」的批号展示规则优化：同产品同日期存在多批号时，显示 `批号1/批号2` 提醒；仅单批号时不显示序号。
- 运单详情标题强调优化：产品与日期保持红色，板数标签红色；同日期多批号时仅批号中“差异字母”标红，相同字母保持默认色。
- 运单详情删除按钮始终可见；已完成订单删除会给出明确不可删提示，避免“无响应”体验。

## 3.0.13 (2026-04-29)

### Changed
- 数据备份页交互简化为“时间快照 + 恢复”，弱化文件级技术信息展示。
- 新增自动备份策略：`关闭`、`每天一次`、`每周一次`。

### Added
- 新增备份管理能力：单条删除快照、批量清理旧快照（保留最近 30 条）。
- 新增快照自动上限裁剪（默认最多保留 90 条）。
- 新增电脑端接收工具（CLI 首版），支持输入连接码从手机发送端拉取备份到电脑。

## 3.0.12 (2026-04-29)

### Changed
- 数据备份页新增备份快照列表，按时间倒序展示本地备份。
- 备份快照支持选择恢复；恢复前会自动备份当前数据库，再用 SQL 覆盖业务表数据。

### Fixed
- 修复重置数据库后 DEBUG/重启再次导入内置库存的问题；内置库存初始化现在只在真正首次安装时执行一次。

## 3.0.11 (2026-04-29)

### Fixed
- 修复数据备份重置/导入完成后重建首页 key，导致 Navigator Overlay 出现重复 GlobalKey 红屏的问题。
- 首页改为在数据库实例变化时刷新统计数据，不再强制重建根 home route，保持路由与 Overlay 树稳定。
- 修正重置数据库语义：重置改为 SQL 级清空业务表，不再删除数据库文件，也不会重新保留内置库存。
- 修正接收导入底层策略：全量接收改为 SQL 覆盖业务表数据，不再删除并替换本机数据库文件。

## 3.0.10 (2026-04-28)

### Fixed
- 修复「重置数据库」后内置库存重新导入导致资料没有归零的问题。
- 数据库接收/重置后的重开流程默认不再执行内置库存初始化；仅首次安装启动仍会自动初始化内置库存。
## 3.0.9 (2026-04-28)

### Fixed
- 修复数据备份接收导入后崩溃：基础资料已有多条产品时，种子数据空库判断不再使用 `getSingleOrNull()` 触发 `Too many elements`。
- 收紧接收导入恢复流程：导入失败后的数据库重开异常不再在 `finally` 中二次抛出导致红屏。
## 3.0.8 (2026-04-28)

### Changed
- 数据备份页完成发送/接收重构：主界面改为「发送」「接收」两个圆形入口，隐藏手动地址输入。
- 发送端生成二维码与 6 位配对码，接收端支持扫码接收或输入配对码自动连接。
- 局域网互传保留 HTTP 数据库传输，新增 UDP 自动发现发送端，降低人工填写 IP 的操作成本。
- 本地备份与重置数据库降级为底部工具区，避免干扰主流程。

### Fixed
- 数据备份读取 Android 真实 sqflite 数据库目录，修复真机生成备份提示找不到数据库的问题。
- 出库日历点击运单不再重置滚动位置，运单出库明细产品与日期拆分显示，减少截断。
## 3.0.7 (2026-04-28)

### Fixed
- 修复首页订单统计口径：`未完成` 现按 `未完成 + 已拣货`（即非完成）统计，避免订单增删改状态后首页数字看似不同步。
- 修复订单状态回退与出库日历不同步：当订单从 `已完成` 回退到 `未完成/已拣货` 时，自动清理该订单出库流水，库存与出库日历同步恢复。

## 3.0.6 (2026-04-28)

### Changed
- 回退新增运单页面误改的按钮样式，恢复为原三按钮布局（结束 / 继续 / 下一单）。
- 恢复新增运单页面分区标题「订单信息」「产品明细」显示，避免信息层级丢失。

## 3.0.5 (2026-04-28)

### Changed
- 出库明细卡片去掉「按运单」文案，产品与日期强制单行显示（超长省略），避免换行错位。
- 新增运单底部按钮重排为「结束 / 继续 / 下一单」；「结束」返回订单信息页。
- 新增运单商家录入改为历史商家下拉选择，收敛大标签占行问题。
- 重复明细从硬拦截改为弹窗确认「是否累加」，确认后合并箱数。
- 引入未完成运单预占库存：产品可用量按预占后计算，预占到 0 的批号不再可选。
- 订单信息快捷标签改为默认「未完成」，顺序调整为「未完成 / 今日 / 昨日 / 一周 / 一月」。
- 订单信息统计增强为「完成 / 未完成 / 已拣货」三项，并增加产品日期维度备货板数汇总文案。
- 首页总库存增加「预占后库存」展示，用于未完成订单的未来库存预判。

## 3.0.4 (2026-04-28)

### Changed
- 出库明细数量列布局优化：箱数与板数保持固定双列对齐，存在与不存在“+箱数”时都保持同列位置。
- 新增运单底部操作优化：新增「结束」按钮，点击后直接退出新增流程并返回首页。

## 3.0.3 (2026-04-28)

### Changed

- 触发 GitHub Actions 重新生成可覆盖安装的发布包。
## 3.0.2 (2026-04-28)

### Added
- 新增 `untitled.pen` 真机 UI 基准稿，补齐 8 张实际页面的可编辑设计参考。

### Changed

- 出库日历出库明细改为按运单分组，商家只在运单卡片头部显示一次并标红。
- 出库明细卡片改为紧凑布局，合计箱数并入运单标题右侧，产品行只展示产品批号、箱数与板数。

## 3.0.1 (2026-04-28)

### Changed
- 库存明细筛选交互优化：快捷标签支持再次点击取消选中，新增 `X` 一键清空筛选。
- 选中产品快捷标签后自动展开该产品所有批号，不再折叠。
- 库存明细列表移除「有库存」标记，仅保留零库存「已空」标记。
- 版本更新策略调整为三段式数字递增（如 `2.0.1 -> 2.0.2`、`2.1.1 -> 2.1.2`），并提供脚本 `scripts/bump_version.ps1` 一键升版。

## 2.1.0+4 (2026-04-28)

### Added
- 新增运单同一运单内「同产品 + 同批号」重复添加拦截与提醒。
- 订单列表状态新增「全部」并默认选中，增加「完成/未完成」数量统计。
- 出库日历支持点击运单后按运单过滤当日出库明细。
- 二维码预览页新增二维码大小滑条，并持久化保存大小设置。

### Changed

- 新增运单 TS 展示改为基于基础资料 TS 勾选结果显示标签。
- 出库日历订单摘要增强为「订单 X单 · Y箱」，并展示运单号、商家、出库箱数。
- 出库明细增强展示运单号与商家信息，便于追溯当日出库来源。
- 出库变化文案调整为「库存变化 -X箱」，且仅在变化大于 0 时显示。
- 日历快捷筛选与「自定义范围」按钮固定单行（窄屏横向滚动，不再自动换行）。

### Fixed
- 修复新增运单产品信息 TS 识别与展示不一致的问题。
- 修复出库日历窄屏下筛选控件换行导致的布局问题。
- 修复二维码预览在较小可视区域的内容溢出问题。
